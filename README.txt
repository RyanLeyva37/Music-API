Music API 

An application for storing and updating a list of musicians, their discography, and music festival dates. 

Description

As a team of three, we worked together to develop a full interface where you could create new artists to the database, update festival dates, update discography, and much more. 


Features

Create and update artists in a database
Input new albums to an artist's discograhy
Update upcoming festival dates
